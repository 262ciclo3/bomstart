# Administracion de Base de Datos I

## Proposito del Curso

Disenar e implementar bases de datos relacionales a partir de los requerimientos del negocio, aplicando modelado conceptual, diseno logico, normalizacion, lenguaje SQL y principios de integridad de datos, garantizando consistencia, mantenibilidad y soporte para aplicaciones web empresariales.

## Producto Final del Curso

**Base de Datos Relacional Implementada y Validada**

Incluye:

* Modelo Entidad-Relacion (ER).
* Modelo logico relacional.
* Diccionario de datos.
* Modelo normalizado.
* Scripts DDL.
* Scripts DML.
* Consultas SQL.
* Integridad referencial implementada.
* Evidencias de funcionamiento.
* Analisis comparativo entre modelo relacional y alternativas NoSQL para el dominio del proyecto.
* Integracion con LP1.
