## II. Sumilla

La asignatura de Administración de Base de Datos I pertenece al área de especialidad de ingeniería de software, es de carácter teórico-práctico y de tipo obligatorio. Tiene como propósito que el estudiante diseñe e implemente bases de datos relacionales a partir de los requerimientos del negocio, aplicando modelado conceptual, diseño lógico, normalización, lenguaje SQL y principios de integridad de datos, garantizando consistencia, mantenibilidad y soporte para aplicaciones web empresariales. Los contenidos principales son: diseño conceptual y lógico de bases de datos, implementación y consulta de bases de datos, e integración del proyecto de base de datos relacional con la aplicación web desarrollada en LP1.
