## II. Sumilla

La asignatura de Administración de Base de Datos I pertenece al área de especialidad de ingeniería de software es de carácter teórico-práctico, de tipo obligatorio. Aporta al perfil de egreso en la competencia general de desarrollo espiritual en la sub competencia de aprendizaje del servicio enfocado en la misión, en la competencia general pensamiento de orden superior y en la competencia específica de gestión y desarrollo de software en el nivel 2. Tiene como propósito que el alumno sea capaz de diseñar correctamente modelos de Base de Datos para cualquier sistema e interactuar con ellos. Los contenidos a tratar son: Modelo conceptual de una base de datos; Diagrama Entidad Relación; Normalización, Diseño e Implementación de una base de datos. Arquitectura y componentes de una base de datos. Desarrollo de Sistemas de Base de Datos de una plataforma Cliente-Servidor.
