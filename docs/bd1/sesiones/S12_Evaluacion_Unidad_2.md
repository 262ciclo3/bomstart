# S12 - Evaluación Unidad 2: implementación y consultas SQL

## 1. Introducción

Tiempo: 20 min.

### 1.1 Propósito

Evaluar scripts DDL, DML, consultas fundamentales, reportes y análisis NoSQL del proyecto.

### 1.2 Resultado de aprendizaje

El estudiante sustenta el producto de datos desarrollado, demuestra evidencias verificables y explica su aporte individual.

### 1.3 Producto de sesión

Producto U2: base implementada con datos, consultas, reportes y comparación NoSQL.

### 1.4 Motivación de la sesión

La evaluación revisa que la base de datos funcione como soporte del sistema web y no como scripts aislados.
