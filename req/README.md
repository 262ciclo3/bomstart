# REQ - Ingenieria de Requerimientos

Carpeta para artefactos del curso **Ingenieria de Requerimientos**.

## Producto Final

**Especificacion de Requerimientos de Software (SRS) Documentada**

## Entregables Previstos

* Identificacion y analisis de stakeholders.
* Alcance del sistema.
* Requerimientos funcionales.
* Requerimientos no funcionales.
* Reglas de negocio.
* Historias de usuario.
* Casos de uso.
* Priorizacion de requerimientos.
* Prototipos de interfaces.
* Matriz de trazabilidad.
* Documento SRS basado en IEEE 29148.
* Validacion y aceptacion de requerimientos.
